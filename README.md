# vue-music-player

一款很简易的音乐播放器

[demo地址](https://gdutwyg.github.io/vue-music-player)


### Project setup(安装依赖)
```
yarn install
```

### Compiles and hot-reloads for development(项目运行)
```
yarn run serve
```

### Compiles and minifies for production(项目打包)
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
### deploy(项目部署)
```
sh deploy.sh
```
